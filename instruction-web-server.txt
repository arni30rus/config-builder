⚙️ Настройка порта и доменного имени

По умолчанию приложение запускается на порту 8800 и доступно по IP-адресу сервера. Если вы хотите изменить порт или привязать приложение к доменному имени (например, config.yoursite.com), внесите изменения перед запуском установки.
🐳 Для установки через Docker Compose

1.    Откройте файл docker-compose.yml в корне проекта.
2.    Изменение порта: Найдите секцию frontend и измените левую цифру в строке ports:
   
    frontend:
      build: ./frontend
      ports:
        - "8800:80" # Замените 8800 на нужный порт (например, "9090:80" или "80:80")
     
     
3.   Настройка домена: Откройте файл frontend/nginx.conf. Найдите строку server_name _; и замените _ на ваше доменное имя:

    server {
        listen 80;
        server_name config.yoursite.com; # Ваш домен
        # ... остальной код
    }
         
(Если вы настраиваете домен, обычно порт меняют на 80:80 в docker-compose.yml, чтобы сайт открывался без указания порта в браузере).

После внесения изменений запустите команду docker compose up -d --build.

🖥️ Для установки через скрипт (Bare-Metal)

1.    Откройте файл config-builder.nginx в корне проекта до запуска скрипта install.sh.

2.    Изменение порта: Найдите строку listen 8800; и измените номер порта:
     
    listen 8800; # Замените 8800 на нужный порт (например, 80 для домена)
     
     
3.    Настройка домена: Найдите строку server_name _; и замените _ на ваше доменное имя:

    server_name config.yoursite.com; # Ваш домен
     
     
После этого запустите sudo ./install.sh. Скрипт автоматически скопирует измененный конфиг в Nginx.
