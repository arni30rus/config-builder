⚡ **Config Builder**

Веб-приложение для генерации конфигураций сетевого оборудования (Cisco, Fortigate, Mikrotik и др.) на основе шаблонов. Позволяет инженерам быстро получать готовые конфиги, заполняя только переменные, и копировать их по блокам для вставки в консоль.

🚀 Основные возможности

   - Шаблонизация Jinja2: Использование переменных {{ hostname }} и циклов.
   - Умные блоки: Разбивка конфига на блоки [BLOCK: Имя] с отдельными кнопками копирования для поэтапной вставки.
   - Опции (Вариативность): Условные блоки (Radio-кнопки). Например, выбор между WAN Static и WAN PPPoE.
   - Переменные: Поддержка локальных (для одного шаблона) и глобальных (общих для всех) переменных.
   - Импорт\экспорт шаблонов.Все шаблоны сохраняются в базе данных PostgreSQL.

🛠 Технологический стек

   - Backend: Python 3.12, FastAPI, SQLAlchemy, Jinja2
   - Frontend: Vue 3
   - Database: PostgreSQL 15
   - Web Server: Nginx (для раздачи статики и проксирования API)

📦 **Установка и запуск**

Есть два способа запуска: через Docker или напрямую на сервере через скрипт установки.

1. **Docker Compose**

 Установите Docker и Docker Compose. [ИНСТРУКЦИЯ ПО  УСТАНОВКЕ](./instruction-docker.txt)
 
 Клонируйте репозиторий:

```bash
  git clone https://github.com/arni30rus/config-builder.git
   cd config-builder
```
Запустите приложение одной командой:
```bash
  docker compose up -d --build
   ```
Приложение будет доступно по адресу: http://<IP-сервера>:8800

2. **Bare-Metal (Ubuntu/Debian)**

Клонируйте репозиторий на сервер.

Запустите скрипт автоматической установки (установит PostgreSQL, Node.js, Python, настроит Nginx и systemd):
```bash
 chmod +x install.sh
  sudo ./install.sh
 ```    
  Запустите сервис:
  ```bash
   sudo systemctl start config-builder
    sudo systemctl restart nginx
  ```   
  Приложение будет доступно по адресу: http://<IP-сервера>:8800

🗑 **Удаление (Bare-Metal)**

Для полной очистки системы (удаление БД, конфигов Nginx и сервиса) используйте скрипт:
```bash
sudo ./uninstall.sh
 ```
 
⚙️ **Как пользоваться**

   - Нажмите «Редактор шаблонов».
   - Создайте шаблон и вставьте базовый конфиг в текстовое поле.
   - Выделите значения мышкой и используйте кнопки:
         «📦 Выделить блок» — создает блок с кнопкой копирования.
         «✏️ Заменить на переменную» — заменяет значение на {{ var }} и добавляет её в список.
   - Добавьте Опции для вариативных частей (например, разные типы WAN).
   - Вернитесь на главную, выберите шаблон, заполните поля и нажмите «Сгенерировать»!
